Java

In Java, our final project was to create a battleship program that we designed and tested for ourselves. We received instructions with three suggested files and how the game should run, the goal was to create reusable code and gain experience in designing a program with multiple classes. One technique used when creating this program was to make a map of the classes and functions needed. This was a key part to make sure all the needed components were included, designing a modular program was done in this process. Another technique used was in debugging, making small changes and fixing the coordinating parts helped keep changes clear.
